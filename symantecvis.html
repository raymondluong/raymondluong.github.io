Machine Network Visualization
Software Engineering Intern at Symantec
Summer 2015
Overview

An interactive tool to visualize the machine network topology.

Description

As one of my projects as a Software Engineering Intern at Symantec, I developed an interactive tool that visualizes the network of machines that supports the developer pipeline for the Norton eBusiness platform. This tool takes an input JSON, reworks the data, and then renders it using D3.js.

Previously, this topology was drawn by hand and needed to be redrawn everytime the network changed. The tool I created allows the network to be visualized dynamically.
