Deployment Pipeline Dashboard
Software Engineering Intern at Symantec
Summer 2015
Overview

An interactive platform to monitor the deployment pipeline.

Description

My main project as a Software Engineering Intern at Symantec was to design and develop the MOD Dashboard, an internal tool on the Monitoring, Operations, and Deployment team of the Norton eBusiness department. Some features include a log view that tracks changes to the version control system, a docker view that displays the state of the docker registry, and a machines view that visualizes the state of each dev environment.

One of the highlights of this project was the opportunity to take ownership and drive the direction of this dashboard. I came in with very limited knowledge of JavaScript and hadn't even heard of Angular. After the first few weeks, I was already rolling out new features and functionalities, and by the end of my internship I had done a complete overhaul of the interface.
